## Projeto
Site institucional estático da BOX BEEF.

## Fonte de verdade
Leia sempre antes de codar:
- /assets/docs/Texto colado.txt
- /assets/docs/Markdown.md colado
- /assets/docs/UIDesign

## Stack obrigatória
- HTML puro
- CSS puro
- JavaScript puro

## Restrições
- Não usar frameworks
- Não usar bibliotecas externas
- Não usar build tools
- Não renomear arquivos existentes sem necessidade
- Não inventar dados pendentes

## Regras de implementação
- Mobile first obrigatório
- HTML semântico
- CSS organizado e legível
- JS mínimo, resiliente e progressivo
- Manter consistência entre todas as páginas
- Mudanças pequenas e fáceis de revisar

## Workflow
- Primeiro analisar contexto
- Depois listar plano curto
- Depois implementar apenas o escopo pedido
- Ao final informar arquivos alterados e resumo técnico

## Definição de pronto
- Semântica correta
- Responsivo
- CTA principal visível
- Sem dependências externas
- Compatível com os documentos em /assets/docs/
